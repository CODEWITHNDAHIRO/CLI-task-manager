# CLI Task Manager

A lightweight command-line task manager written in Python. Manage your tasks directly from the terminal — no dependencies required.

## What does it do?

This application allows users to add, view, delete, and mark tasks as completed through an interactive command-line menu. All tasks are managed in-memory during the session.

## Features

- View all tasks with completion status
- Add new tasks
- Delete existing tasks
- Toggle tasks as completed / not completed
- Simple numbered menu interface
